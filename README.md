# Project ToDo List


### Core Features


- [x] Add Task
    - Enter a task and add it by clicking Add or pressing Enter.
- [x] Prevent empty tasks.
- [x] View Tasks.
    - Display all added tasks in a list.
    - Show a message when no tasks exist.
- [x] Complete Task
    - Mark a task as completed using a checkbox.
    - Completed tasks should have a strikethrough style.
- [x] Delete Task
    - Remove a task from the list.
- [x] Task Counter
    - Display: 
        - Total Tasks
        - Completed Tasks
        - Remaining Tasks

- [x] Save Data
    - Store tasks in Local Storage so they persiss after page refresh.
